# Enumclaw Forest Estate

Marketing website for 23670 SE 448th St, Enumclaw, WA.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm start
```

## Deployment

Import the repository into Vercel and point `enumclawforestestate.com` to the generated Vercel project.
